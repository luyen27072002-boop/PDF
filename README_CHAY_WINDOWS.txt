CHẠY NHANH TRÊN WINDOWS
========================

1) Giải nén ZIP.
2) Mở thư mục vừa giải nén.
3) Bấm đúp START_WINDOWS.bat
4) Mở Chrome: http://127.0.0.1:8000

Nếu đang có server cũ chiếm port 8000, đóng terminal cũ bằng Ctrl+C hoặc sửa run.bat sang port 8001.

CÁCH SỬA TEXT MÀ KHÔNG MẤT NỀN
===============================

1) Upload PDF.
2) Chọn Edit text.
3) Để các tùy chọn:
   - Xóa gốc: Xóa text, giữ nền
   - Preview sạch: bật
   - Ẩn khung: bật
   - Xuất kiểu: Native: xóa chữ giữ nền
   - Giữ kiểu chữ gốc: bật nếu muốn lấy font/cỡ/màu từ chữ cũ
4) Click trực tiếp vào chữ cũ.
5) Gõ chữ mới.
6) Chỉnh Font / Size / Màu / B / I / Căn trên thanh trên.
7) Bấm Xuất PDF.

Bản Native sẽ cố gắng xóa riêng glyph chữ cũ khỏi PDF, không tô ô trắng, không che watermark, không che đường kẻ hay nền bên dưới.
Nếu file PDF là ảnh scan không có text layer thì không thể sửa chữ thật nếu chưa thêm OCR.
